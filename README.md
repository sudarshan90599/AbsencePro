A web-based AbsencePro built using **Django**, designed for educational institutions to manage student leave requests efficiently.  
Students can submit leave applications online, and mentors/directors can review, approve, or reject them seamlessly.

---

##  Features
- Student registration and profile management  
- Leave request submission and tracking  
- Mentor approval/rejection workflow  
- Director final approval system  
- Role-based authentication and dashboards  
- Responsive UI built with Bootstrap  

---

##  Tech Stack
- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** Django (Python)  
- **Database:** SQLite  
- **Version Control:** Git & GitHub  

---

## Installation
# Clone the repository
git clone https://github.com/sudarshan90599/AbsencePro.git

# Navigate into the project folder
cd leave-management
cd leave_project

# Install dependencies
pip install -r requirements.txt

# Apply migrations
python manage.py migrate

# Run the server
python manage.py runserver
